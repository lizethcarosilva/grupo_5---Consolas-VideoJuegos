Conoce al Equipo Detrás de Console Wars

Apasionados por el desarrollo web, la tecnología y el universo gamer.

¿Por qué creamos este sitio web?

Somos el Grupo 5, un equipo de estudiantes apasionados por el desarrollo web y los videojuegos. Creamos este sitio como parte del laboratorio de maquetación HTML, CSS y control de versiones con GitHub utilizando la metodología SCRUM.


Nuestra Misión

Brindar análisis objetivos, comparativas de rendimiento, catálogo de juegos e información imparcial para ayudar a la comunidad gamer a decidir entre las dos gigantes del mercado: PlayStation y Xbox.
